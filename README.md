# Payload CMS App

A simple Payload CMS 3 app built with Next.js and SQLite.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000/admin](http://localhost:3000/admin)

## Collections

- **Users** — admin users
- **Media** — file uploads
- **Content** — text content (title + rich text)

```
GET http://localhost:3000/api/content
```
